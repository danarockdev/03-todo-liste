# 📝 Meine To-Do Liste

Eine einfache und übersichtliche To-Do-Listen-App, entwickelt mit HTML, CSS und JavaScript. Das Projekt nutzt eine moderne Karten-Ansicht (`card`), um Aufgaben elegant darzustellen.

## 🚀 Funktionen

* **Aufgaben hinzufügen:** Tippe eine Aufgabe ein und klicke auf "Hinzufügen" oder drücke die Enter-Taste.
* **Aufgaben löschen:** Entferne erledigte Aufgaben im Handumdrehen mit dem "X"-Button.
* **Responsive Design:** Die App passt sich optimal an verschiedene Bildschirmgrößen an.

## 🛠️ Technologien

* **HTML5:** Struktur und semantischer Aufbau (inklusive Card-Layout).
* **CSS3:** Modernes, zentriertes Design mit Schatten-Effekten.
* **JavaScript (Vanilla):** Dynamische DOM-Manipulation für die App-Logik.

## 💻 Installation & Start

Du benötigst keinen Server, um dieses Projekt zu starten.

1. **Klonen oder Herunterladen:** Lade die Dateien (`index.html`, `style.css`, `script.js`) in einen Ordner auf deinem Computer.
2. **Öffnen:** Klicke doppelt auf die Datei `index.html`, um die App direkt im Webbrowser zu öffnen.
3. *Optional (Visual Studio Code):* Nutze die Erweiterung **Live Server**, um die App mit automatischem Neuladen zu starten.
